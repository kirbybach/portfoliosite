# kirbybach.com

Personal portfolio site built with React, TypeScript, and Vite.

## Pages

- **Home** — Hero, about, experience, and contact sections
- **More** — Hub linking to reading list and project writeups
- **Reading** — Bookshelf of current and finished reads
- **Projects** — Detailed project writeups and demos

## Tech Stack

- React 19 + TypeScript
- React Router v7
- Vite
- Vanilla CSS

## Development

```bash
npm install
npm run dev
```

## Production

```bash
npm run build
npm run preview
```
